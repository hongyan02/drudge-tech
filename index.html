常见问题 / FAQ

1) 数据丢失或显示异常 / Missing or incorrect data

请提供发生时间点和具体页面；若可复现，按步骤说明更易定位。 Provide the time it happened and the exact screen; reproducible steps help troubleshooting.

2) 工时阶梯配置问题 / Tier configuration issues

工时区间必须从0开始，且相邻区间必须无缝连接。系统会自动验证配置的合法性。 Hour ranges must start from 0 and adjacent tiers must connect seamlessly. The system validates configuration automatically.

3) 加班工资计算不正确 / Incorrect overtime pay calculation

请检查工时阶梯配置是否正确，确保所有区间连续且无重叠。系统按总工时匹配对应的阶梯计算工资。 Please verify tier configuration is correct - ensure all ranges are continuous without overlaps. The system calculates pay based on total hours matching the appropriate tier.
